Technology
核心技术

Agent 分身建模（Agent Twin）

基于 SprixFoundation Model 与 SprixAgent OS，融合用户技能、履历、AI 面试、多轮交互及任务执行数据，构建可持续学习的 Agent 分身，实现能力建模、行为建模与决策建模，复现用户在真实工作场景中的执行方式。行为一致性 BSS ≥ 60%，任务完成率 TCR ≥ 90%。

多 Agent 智能调度引擎

面向 Human-Agent Workforce 的智能调度引擎，基于 SprixMarket Engine（贝叶斯-Stackelberg 市场机制）与多 Agent 协同框架，自动完成任务拆解、团队组建、资源配置与协同执行。任务成功率 TSR ≥ 92%，任务周期缩短 TCT ≥ 55%，成本降低率 CRR ≥ 38%。

Human-Agent 协作评估体系

构建行业领先的 Human-Agent 双重质量评估体系，通过 Rubric 规则库、LLM Judge 与专家反馈闭环，实现任务结果质量评估、风险预警与 Agent 信誉沉淀。量化评估候选人的 Human-Agent 协同生产力，专家一致率 EAR ≥ 90%。
